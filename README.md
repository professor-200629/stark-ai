# STARK — Personal AI Operating System

A voice-controlled AI assistant built with Python + Gemini.

## Features
- Voice controlled — speaks and listens continuously
- Controls PC — volume, brightness, screenshots
- WhatsApp messaging and calls by voice
- Real GPS location — finds nearby places
- Screen reading and image identification
- RAG agent with web search
- Travel planning, flight search, weather anything

## Setup
```
pip install -r requirements.txt
python stark_terminal_ui.py
```

## Testing
1. Run `python stark_terminal_ui.py`
2. Say "hello STARK"
3. Say "what time is it"
